# Contributing

This toolkit is a starting point for the community. Contributions welcome!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-improvement`)
3. Make your changes
4. Run the scripts against a test environment
5. Submit a pull request

## Areas for Contribution

- Support for additional botcomponent types (evaluation cases, topics)
- VS Code extension wrapper that calls these scripts automatically
- Multi-agent export (export all agents from an environment)
- Connection reference remapping (auto-fix broken connection refs after push)
- CI/CD pipeline integration (GitHub Actions workflow for automated push)
- Dry-run mode (`-WhatIf` support) for all scripts

## Testing

Before submitting, verify your changes against a CDX or trial Power Platform environment.
